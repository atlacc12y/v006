  1. เปิด Apps Script ของโปรเจกต์นี้
  2. คลิกไอคอนรูปนาฬิกา "Triggers" ที่เมนูซ้าย
  3. กด "+ Add Trigger" มุมขวาล่าง

  5. ช่อง "Select event source" เลือก Time-driven
  6. ช่อง "Select type of time based trigger" เลือก Day timer
  7. เลือกช่วงเวลา (แนะนำ 23:00 to 24:00 น. เพื่อเก็บตัวเลขสรุปตอนปลายวัน) แล้วกด Save
